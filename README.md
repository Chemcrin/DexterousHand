# 灵巧手工程

> 基于 Python NumPy + Matplotlib 3D 的仿生机械灵巧手运动学仿真系统。

---

## 项目简介

本项目构建了一个**严格遵循人类手部解剖结构**的机械灵巧手 3D 运动学仿真系统。通过纯 NumPy 数值计算定义关节/骨骼/运动学关系，使用 Matplotlib 3D 进行可视化渲染，以点表示关节、以线表示骨骼，实现多种仿生手势与动作。

### 核心特点

- 纯运动学仿真，基于关节角度的正向运动学坐标变换
- 1:1 仿生人类手部结构（含 CMC 虎口关节、可活动腕关节）
- 13 种以上仿生动作，支持平滑过渡和循环播放
- 轻量依赖，仅需 `numpy` + `matplotlib`

---

## 手部结构

机械灵巧手的仿生结构完全匹配真实人手：

```
FOREARM_BASE → WRIST → 五指
```

| 手指 | 关节链 | 自由度 |
|------|--------|:------:|
| 拇指 | CMC(3DOF) → MCP(2DOF) → IP(1DOF) | 6 |
| 食指 | MCP(2DOF) → PIP(1DOF) → DIP(1DOF) | 4 |
| 中指 | MCP(2DOF) → PIP(1DOF) → DIP(1DOF) | 4 |
| 无名指 | MCP(2DOF) → PIP(1DOF) → DIP(1DOF) | 4 |
| 小指 | MCP(2DOF) → PIP(1DOF) → DIP(1DOF) | 4 |
| 腕部 | WRIST(3DOF) | 3 |

### 坐标系（右手系）

- 原点：WRIST 腕关节中心
- +X：拇指 → 小指方向
- +Y：腕部 → 指尖方向
- +Z：垂直手掌平面（屈伸方向）

## 已实现的仿生动作

### 基础动作
| 动作 | 说明 |
|------|------|
| 摊开手掌 | 初始姿态，所有关节伸直 |
| OK手势 | 拇指 + 食指指尖相触 |
| 精准抓握 | 拇指对掌，四指轻曲 |
| 握拳 | 所有手指完全屈曲 |
| 竖大拇指 | 拇指伸直上翘，四指屈曲 |

### 数字手势
| 动作 | 说明 |
|------|------|
| 数字1 | 仅食指伸直 |
| 数字2 | 食指 + 中指伸直 |
| 数字3 | 食指 + 中指 + 无名指伸直 |
| 数字5 | 全手掌展开 |

### 扩展动作
| 动作 | 说明 |
|------|------|
| 竖小拇指 | 小指伸直，其余弯曲 |
| 半握拳 | 手指中度弯曲 |
| 手指依次屈伸 | 食→中→无名→小指依次弯曲/伸直 |
| 拇指对掌 | 拇指从外侧旋转触碰各指尖 |
| 挥手 | 腕部驱动，连带五指晃动 |

## 技术栈

- **语言：** Python 3.12
- **计算：** NumPy（旋转矩阵、欧拉角变换）
- **可视化：** Matplotlib 3D (`mpl_toolkits.mplot3d`)
- **无物理引擎**，纯运动学仿真

---

## 快速开始

```bash
# 安装依赖
pip install numpy matplotlib

# 运行仿真
python dexterous_hand_sim_fixedX.py
```

3D 视图中可旋转、缩放、平移视角，动作将按顺序自动播放。

## 设计修正

相比原始需求，本项目在以下方面进行了仿生学修正：

1. **拇指增加 CMC 关节**：虎口关节具备 3 自由度，保证对掌、OK、捏取等动作自然
2. **腕部可活动**：增加前臂参考段（FOREARM_BASE），腕关节具备 3 自由度
3. **手指长度真实化**：中指最长，小指最短，食指与无名指长度接近
4. **关节定位修正**：杜绝关节黏连、拇指内凹等前置版本出现的问题
